# Akira Landing Page

Landing page para o Akira — IA para investidores. Site estático em HTML/CSS/JS puro, sem dependências de build.

## Estrutura

```
index.html       Marcação da página
css/style.css     Estilos
js/script.js      Conteúdo dinâmico (cards, passos, FAQ) e interação do acordeão
assets/           Imagens (logo, mascote)
```

## Rodando localmente

Basta abrir `index.html` em um navegador, ou servir a pasta com qualquer servidor estático, por exemplo:

```
npx serve .
```
